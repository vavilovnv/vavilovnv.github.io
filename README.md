
# vavilovnv.github.io

Personal static website for GitHub Pages.
This repository contains the source for my personal business-card website.
It presents my backend engineering profile, technology stack, contact links, and bilingual EN/RU content.

## Structure

- `index.html` - root entry point required by GitHub Pages.
- `src/styles.css` - page styles.
- `src/main.js` - theme, language switcher, copy email, and reveal behavior.
- `assets/icon.png` - site icon.

No build step is required.
